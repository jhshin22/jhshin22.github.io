# 전략계급전

8x8 보드에서 진행되는 2인용 비공개 정보 전략 보드게임입니다.

## 실행

GitHub Pages에서 `Chu_war/index.html`을 엽니다.

## 포함 기능

- 상단/하단 플레이어 수동 배치
- 전략 랜덤 배치와 배치 초기화
- 턴 전환 가림 화면
- 자기 기물만 정체 표시, 상대 비공개 기물은 ? 표시
- 기본 이동, 기병 직선 이동
- 1성장군/2성장군/3성장군 특수행마
- 4성장군 비공개 공격 승리 후 복귀
- 스파이 공개 여부 기반 전투 판정
- 폭탄병 이동 공개 및 전투 시 동시 제거
- 왕 제거 및 왕만 남는 승리/무승부 판정
- 사람 vs 사람 / 사람 vs AI 쉬움 / 사람 vs AI 보통 / 사람 vs 봄 / 사람 vs 여름
- 경기 종료 시 브라우저에 JSON 경기 로그 자동 저장

## 배치 방식

플레이어가 누르는 랜덤 배치는 완전 무작위가 아니라 5개 컨셉 중 하나를 고르는 전략 랜덤 배치입니다. 컨셉 핵심 기물만 먼저 배치하고, 나머지 기물은 랜덤으로 채웁니다. 하단 플레이어는 같은 컨셉을 아래쪽 진영 기준으로 뒤집어 적용합니다.

- 왕 요새형
- 보병 정찰선형
- 전방 차단형
- 1성 기습형
- 고급장군 전진압박형

## AI 모드

AI는 상단 플레이어를 담당합니다.

- 쉬움: 합법 수 중 무작위 선택을 기본으로 하되, 공격 가능한 수를 가끔 우선합니다.
- 보통: 공개 기물과 비공개 후보군 기대값을 이용해 수를 평가합니다. 기병 장거리 공개, 왕 이동, 특수행마 공개 손해를 감점합니다.
- 봄: 즉시 승리, 왕 방어, 공개 기물 상성, 비공개 스파이/폭탄병 위험도, 패스 후보를 함께 점수화합니다. 비공개 상대 기물의 실제 정체는 보지 않고 공개 정보와 후보군 추정만 사용합니다.
- 여름: 비공개 상대 기물을 왕 후보로 점수화하고, 상대 이동 기록을 기억하며, 성향별 가중치로 왕 후보 추적·정보 공격·왕 방어를 다르게 운영합니다.

## 경기 로그

경기가 종료되면 브라우저 localStorage에 최근 30개 경기 로그가 JSON 형식으로 저장됩니다.

- 초기 배치 보드
- 각 턴의 이동/공격/패스 기록
- 이동 전 기물과 대상 기물의 실제 정보
- 이동 후 보드 상태
- 양쪽 잔존 기물 수
- 최종 보드
- 승패/무승부 결과
- 화면에 표시된 최근 기록

사이드 패널의 경기 로그 영역에서 최근 로그를 클립보드에 복사하거나 JSON 파일로 다운로드할 수 있습니다. 새 로그에서는 보드 위 기물에 `onBoard: true`와 `alive: true`를 함께 기록해, 배치용 pool 상태와 실제 보드 생존 상태가 혼동되지 않도록 했습니다.

## 봄 AI 배치 전략

봄 모드는 기본 5개에 전용 5개를 더해 총 10개 배치 컨셉 중 하나를 사용합니다.

기본 5개:

- 왕 요새형
- 보병 정찰선형
- 전방 차단형
- 1성 기습형
- 고급장군 전진압박형

봄 전용 추가 5개:

- 스파이 대기형
- 기병 라인장악형
- 중앙 돌파형
- 장군 유도형
- 왕 위장 분산형

## 여름 AI 배치 전략

여름 모드는 완전 랜덤 배치 대신 왕 방어가 비교적 안정적인 5개 컨셉 중 하나를 사용합니다.

- 왕 요새형
- 보병 정찰선형
- 전방 차단형
- 스파이 대기형
- 왕 위장 분산형

## 여름 AI 1차 구현

여름 AI는 봄과 다른 접근법으로 만든 정보전형 AI입니다.

- 비공개 상대 기물을 왕 후보로 보고 위치 기반 후보 점수를 계산합니다.
- 후방에 남은 비공개 기물, 중앙에 있는 비공개 기물, 주변에 보호 기물이 있는 비공개 기물을 더 유력한 후보로 봅니다.
- 후반에는 가장 유력한 왕 후보 쪽으로 접근하는 수를 크게 선호합니다.
- 공개 기물끼리의 상성상 손해 전투를 피합니다.
- 자기 공개 기물이 공개 상대 기물에게 잡히는 위험을 줄이는 수를 선호합니다.
- 패스 후보를 별도로 넣지 않아, 합법 수가 있으면 기본적으로 움직입니다.

## 여름 AI 2차 구현

- 자기 왕 주변의 호위 기물과 적 접근 위험을 별도로 계산해 왕 방어를 강화합니다.
- 상대 이동 기록을 기억해 비공개 기물별 왕 후보 점수를 갱신합니다.
- 후방에서 오래 움직이지 않는 비공개 기물은 왕 후보 점수를 올립니다.
- 전방으로 과감히 이동한 비공개 기물은 왕 후보 점수를 낮춥니다.
- 공개된 기물은 공개 정보 기준으로 후보를 확정합니다.
- 장거리 이동, 1성 특수행마, 2성 특수행마, 3성 특수행마 형태를 감지해 후보를 확정합니다.
- 균형형, 정찰형, 압박형, 수비형, 함정형 5개 성향 중 하나를 매 게임 선택합니다.
- 성향에 따라 왕 후보 추적, 정보 공격, 공개 기물 회피, 왕 방어 가중치를 다르게 적용합니다.

## AI 로직 검토 반영

- 여름 AI도 전략 배치를 사용하도록 수정해 초반 왕 방어 붕괴 가능성을 낮췄습니다.
- 여름 AI의 가상 전투 계산에서 폭탄병이 비공개 기물을 공격하는 경우를 실제 룰처럼 동시 제거로 반영했습니다.
- 여름 AI의 가상 전투 계산에서 4성장군 비공개 공격 승리 후 복귀 가능성을 반영했습니다.
- 왕 주변 호위 기물이 이탈하는 수에 별도 감점을 추가했습니다.
- 왕 주변에 가까운 호위 기물의 가치를 더 크게 평가하도록 조정했습니다.
- AI 턴에서 비공격 특수행마가 단순 이동으로 쓰이는 경우를 차단하는 가드 로직을 추가했습니다.
- 비공격 특수행마는 공개 위협 회피이거나, 전선에서 다음 공격 거리를 만드는 경우에만 후보로 유지합니다.
- AI의 초반 기병 2칸 이상 장거리 이동과 비공개 기물 공격을 원칙적으로 억제했습니다.
- 초반 기병 돌격은 공개 스파이 제거, 마지막 비왕 기물 제거, 강한 왕 후보 압박, 안전한 라인 장악일 때만 예외적으로 허용합니다.
- 중후반에는 기병 장거리 이동 제한을 완화해 추격과 라인 장악 가치를 살립니다.
- 여름 AI의 정찰형 정보공격 가중치를 낮춰 무리한 비공개 공격 빈도를 줄였습니다.
- 여름 AI가 고급 장군으로 왕 후보가 약한 비공개 기물을 공격하는 선택을 강하게 감점합니다.
- 목표가 약한 빈칸 이동, 공개 기물의 무목적 이동, 왕 주변 호위 붕괴 이동에 추가 감점을 넣었습니다.
- 공개 기물끼리 동급 교환을 과대평가하지 않도록 무승부 전투 점수를 낮췄습니다.
- 여름 AI가 왕 주변에 무조건 밀집하는 경향을 줄이고, 안전한 빈 도주칸을 확보하도록 왕 방어 평가를 수정했습니다.
- 왕 인접 호위의 가치는 낮추고, 왕에서 2칸 떨어진 차단선과 열린 통로의 가치를 높였습니다.
- 왕의 안전한 도주칸이 0~1개뿐이면 큰 감점을 주고, 아군 기물이 왕을 둘러싸 도주로를 막는 경우도 감점합니다.
- 로그 분석에서 확인된 폭탄병 비공격 이동, 초중반 기병의 비공개 기물 공격, 비공격 특수행마 낭비를 추가 정책 가드로 제한했습니다.
- 두 번째 로그 분석에서 확인된 기병 빈칸 장거리 돌입을 추가 제한했습니다.
- 공개 기물이 직전 이동을 그대로 되돌리는 반복 왕복을 기억해 차단합니다.
- 왕이 공개 기병·공개 장군에게 직접 위협받는 상황에서는 왕 위험을 낮추는 수를 우선하도록 긴급 가드를 강화했습니다.
- 고급 장군이 왕 후보 점수가 낮은 비공개 기물을 공격해 폭탄과 교환되는 선택을 더 강하게 제한했습니다.
- 왕 앞 차단 기물이 움직이면 즉시 공개 기병·공개 장군 라인이 열리는 수를 금지했습니다.
- 왕이 이미 직접 위협받는 경우, 위협을 줄이지 못하는 공개 기물 공격이나 우회 이동을 차단합니다.
- 후반이라도 기병 장거리 빈칸 이동은 왕 후보 압박, 라인 압박, 왕 방어 개선이 없으면 제한합니다.
- 새 게임 시작 시 AI 정책 가드의 직전 이동 기억을 초기화합니다.
- 후반에 하단 비공개 기물이 1개뿐이면 사실상 왕 후보 확정으로 보고, 닿을 수 있는 상단 비왕 기물이 즉시 공격합니다.
- 왕 후보 확정 공격은 기존 점수 평가보다 우선하는 긴급 행동으로 처리합니다.
- 여름 AI 스케줄에 턴 잠금을 추가해, 전투 알림이나 비동기 타이머 때문에 같은 상단 턴에서 여러 AI 행동이 연속 실행되지 않도록 했습니다.
- 기병이 이미 공개된 뒤에는 장거리 이동을 공개 리스크로 보지 않고, 추격·라인 장악·왕 방어 재배치에 더 적극적으로 사용합니다.
- 기병이 장거리로 즉시 공격할 수 있는 상황에서는 공격 대상 앞 빈칸으로 이동해 다음 턴 공격을 기다리는 수를 차단합니다.
- 여름 AI는 가치 있는 기병 장거리 공격이 있으면 일반 평가보다 먼저 `여름 AI 기병 직격` 행동으로 처리합니다.

## 제외한 2차 기능

저장/불러오기, 되돌리기, 온라인 멀티플레이, 무승부 합의 버튼은 2차 개발 대상으로 남겼습니다.
