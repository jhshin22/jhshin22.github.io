# 전략계급전

8x8 보드에서 진행되는 2인용 비공개 정보 전략 보드게임입니다.

## 실행

GitHub Pages에서 `Chu_war/index.html`을 엽니다.

## 포함 기능

- 상단/하단 플레이어 수동 배치
- 랜덤 배치와 배치 초기화
- 턴 전환 가림 화면
- 자기 기물만 정체 표시, 상대 비공개 기물은 ? 표시
- 기본 이동, 기병 직선 이동
- 2성장군/3성장군/4성장군 특수행마
- 전투 판정, 폭탄병 동시 제거, 1성장군 복귀
- 왕 제거 및 왕만 남는 승리/무승부 판정

## 제외한 2차 기능

저장/불러오기, 되돌리기, AI, 온라인 멀티플레이, 무승부 합의 버튼은 2차 개발 대상으로 남겼습니다.
